## How to Contribute

Want to contribute?

1. **Check the [Roadmap](./Docs/ROADMAP.md)** - See what features need work
2. **Pick an issue** - Look for `good first issue` or `help wanted` tags
3. **Submit a PR** - Follow our [Contributing Guidelines](./README.md#contributing)
4. **Write tests** - We aim for 80%+ coverage
5. **Document your work** - Update relevant docs in `./Docs/`

### Areas we need help

- Testing (unit, integration, E2E)
- Documentation improvements
- Feature implementation (check ROADMAP.md)
- Bug fixes
- Performance optimizations
- Notification enhancements
- Template improvements
